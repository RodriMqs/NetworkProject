# Network Management 

## Objetivos

- Praticar conceitos de POO em Java
- Simular um sistema básico de gestão de rede
- Trabalhar com listas de objetos (`ArrayList`)
- Preparar base para integração com aplicações web
---

## Funcionalidades

- Criar routers com:
  - Nome
  - Endereço IP
  - Número de portas

- Criar switchs com:
  - Nome
  - Endereço IP
  - Número de portas
